배경색 : background-color

name, rgb, rgba, 16진수값으로 줄 수 있다.

linear-gradient

  • 형식 linear-gradient( [angle] , [color-stop] )
  • 각도 : "deg"단위의 값이나 "to키워드" 또는 0% [백분율]와 같은 형식으로 지정할 수 있습니다. 키워드 : top, bottom, left, right로 to top right와 같이 상하 방향과 좌우 방향을 조합하여 사용할 수 있습니다.

linear / 색상만 지정하면 색상변환점 자동으로 배분

linear / 각도45도

linear / 키워드 left

linear / 각도45도 / 색상변환점 80%

radial-gradient

  • 형식 radial-gradient( [shape] [size] [position], [color-stop] )
  • shape:circle(원형), ellipse(타원, 기본)
  • size:원형 그라디언트의 크기(원형의 경우 지름을 지정, 타원의 경우 2개의 길이값, 또는 백분율)
  • position:원형 그라디언트의 중심점의 위치 at[X좌표 Y좌표] 또는 at[키워드]형식으로 지정한다.

circle / 색상만 지정하면 색상변환점 자동으로 배분

circle / at[X좌표 Y좌표]

ellipse / at[X좌표 Y좌표]

background-attachment : 배경이미지 부착 방법

  • scroll : 배경이 요소와 같이 스크롤된다
  • fixed : 배경이 뷰포트에 대하여 고정된다

background-size : 배경 이미지의 크기

  • px단위로 지정할 수 있습니다.
  • %지정 : 부모값이 기준이 됩니다.
  • cover : 배경 이미지의 원래 비율을 무시하고 요소 박스 영역 전체에 배경 이미지가 가득 차도록 합니다.
  • contain : 배경 이미지의 원래 비율을 유지한 채 요소 박스 영역 전체에 배경 이미지가 가득 차도록 합니다.
