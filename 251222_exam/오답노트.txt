

1. 시험문제를 제대로 읽지 않고 문제를 풀었다 >> 틀린 문제 5개중 4개는 문제만 읽어도 맞출 수 있었다.

2. 빈 링크에는 반드시 항상 "#" 임시 링크를 넣어줄 것 (href="#")

3. 로그인폼/입력 작성할 때는 습관적으로 왠만해선 required를 넣어줄 것

4. 자동 조명 기능인 autofocus는 input/textarea/select/button 에 사용할 수 있다. 추가해주면 페이지 로드 시 자동으로 강조시켜준다.

5. (중요!) 종속선택자는 자식선택자와는 다르다. 부모 요소를 불러오는게 아니라, 클래스 이름이 동일한게 여러개 있을 때, 해당 클래스의
타입 종류를 구별해서 특정 클래스만 호출하고 싶을 때 사용한다. (예: <div class="name"> <p class="name">에서 p인 name을 불러오고 싶을 때
p.name 이렇게 호출해준다.)
